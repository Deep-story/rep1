Задача: нужно хранилку IBM FlashSystem 5035 добавить в zabbix мониторинг, но использовать не ручной шаблон настроки, а автоматический(динамический). Автообнаружение, что бы новые разделы или диски добавлялись автоматически.

Т.е что бы zabbix обращался к rest api данной хранилки и брал обновления или автообнаруживал изменения.


Посмотреть про автообнаружение и поискать готовй шаблон на эту хранилку.

Storage-DS3524
Storage-MD3600


Необходимо разобрать:
как работает zabbix
что такое rest api
что такое snmp протокол
что такое json формат обмена данными


api это какой-то интерфейс 

rest api это апи, которое принимает(GET) данные от сервера или отдаем их ему

json это формат хранения данных в текстовом файле, в формате ключ:значение

Сущность которая отдает данные, какому-то серверу в нашем случае по протоколу snmp называет агент.

А устройство которое запрашивает эти данные наз-ся snmp manager

По snmp можно не только получать данные, но и управлять устройством

Используются порты udp 161 на прием,162 на ответ

MIB база данных SNMP

rest api работает поверх http
